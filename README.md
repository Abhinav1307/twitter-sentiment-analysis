# twitter-sentiment-analysis
R shiny web application to scrape tweets based on user-defined search keyword and perform sentiment analysis of the tweets. Sentiment analysis of tweets consists of classifying tweets into emotion classes (i.e., anger, disgust, fear, joy, sadness and surprise) and also polarity classes (i.e., negative, neutral and positive) using na√Øve Bayes classifier. The tweets are scraped, classified into sentiment classes and visualized in R using twitteR, sentiment and ggplot2 packages, respectively.
